REST API

JSON over HTTPS, stateless, and callable from any language or platform. Actively developed with new features and endpoints.

  • 121 endpoints across 16 resource groups
  • Equivalent coverage to the SOAP API
  • X-API-Key header or Bearer JWT authentication
  • Base URL: rest.ecgrid.io

v4.1 — Established

ECGridOS SOAP API

Full WSDL-based SOAP API with equivalent coverage across all platform resources. Stable and production-ready.

  • 242 endpoints for fine grain control
  • XML/SOAP over HTTPS
  • SessionID authentication (Login / Logout)
  • Base URL: os.ecgrid.io/v4.1/prod/ECGridOS.asmx

Platform Capabilities

ECGrid functions as an API-first EDI communication switch serving ISVs, SaaS platforms, and integration providers.

📦Mailbox Management

Create, configure, and manage trading partner mailboxes programmatically. No support tickets, no waiting.

📡Document Tracking

Track every interchange end-to-end. Eliminate blind spots with live status, delivery receipts, and event callbacks.

🔗Protocol Support

AS2, SFTP, FTPS, OFTP, X.400, ebXML, and X12.56 VAN interconnects — unified behind a single API.

🔄Format Flexibility

Route X12, EDIFACT, XML, JSON, and custom formats between trading partners without conversion constraints.

🏗️Platform Integration

Brand the experience as your own. Embed provisioning, monitoring, and operations directly into your platform UI.

⚡Automation

Define workflows triggered by document arrival, error conditions, or trading partner events via callbacks.
